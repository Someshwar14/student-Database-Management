# Student Database Management System

A simple CLI-based Student Management System built using Python and PostgreSQL.

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## Technologies Used
- Python
- PostgreSQL
- psycopg2

## Setup Instructions

1. Install PostgreSQL
2. Create database:
   CREATE DATABASE studentdb;
3. Install dependencies:
   pip install -r requirements.txt
4. Run:
   python app.py
